Mobile apps


Bibles for Android

To install the Pashto or Dari Bible on your Android phone, search for "PashtoBible" or "DariBible" on Google Play Store, or download it from below and install it using the APK file.

  • Pashto APK
  • Dari APK

Pashto Bible on Symbian

To install the Pashto Injil on your Symbian phone, install the "PashtoBible" application from Ovi / Nokia store, or the Opera store.

Opera Mobile Store *

Java Bible (Nokia)

To install the Pashto Injil or Dari Bible on your Nokia Series40 or similar phone, get the "PashtoBible" and "DariBible" MIDP applications by downloading the JAR file. Some phones also need the JAD file.

  • Pashto JAR
  • Pashto JAD
  • Dari JAR
  • Dari JAD

 

N9 / N950 Meego Pashto Bible

To install the Pashto Injil on your N9 series phone, install the "PashtoBible" application from Ovi / Nokia store, or download it:

  • DEB

Blackberry and Playbook Bible

To install the Pashto Injeel or Dari Bible on your Blackberry phone, download the Pashto or Dari Bible BAR file below and install it using DDPB:

  • Pashto BAR
  • Dari BAR
