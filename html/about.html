About the Dove Stream

The Dove Stream is a portable wireless device to share digital media and other resources to users of wifi enabled phones, tablets and laptops.

 

Users accessing files on the Dove Stream are not able to access the internet, as there is no connection other than the wifi link to the Dove Stream.  No user information is required or stored on the Dove Stream.  

 

It simply provides a free wifi network to enable access to the shared files. This makes the Dove Stream ideal for sharing files privately and securely. No login is required, no interaction with anyone is required.  

Thanks

The Dove Stream was adapted from the BibleBox which was conceived by Campbell Smythe and would not have been possible without the foundations of the opensource code behind Librarybox developed by David Darts, Jason Griffey and Matthias Strubel.

Statistics

Dove Stream is designed to be safe and secure. No logins are required and no user data is logged. The system is purposely not connected to the Internet in order to prevent tracking and preserve user privacy.
